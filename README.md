# Amit Computer Education - Online Tuition Website

A modern, responsive website for Amit Computer Education, providing online tuition services for computer subjects to school students from various boards including CBSE, ICSE, State Board, and specialized courses in Web Development.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean, attractive design with gradients and animations
- **Course Listings**: Detailed information about available courses
- **Contact Information**: Phone and email details for direct communication
- **WhatsApp Integration**: Floating WhatsApp button for instant messaging
- **Smooth Scrolling**: Enhanced navigation experience
- **Interactive Elements**: Hover effects and scroll animations

## Courses Offered

- CBSE Computer Science (Classes 9-12)
- ICSE Computer Applications
- State Board Informatics
- Web Development (HTML, CSS, JavaScript, etc.)
- Advanced Programming (Python, C++, Data Science)
- Digital Literacy

## Technologies Used

- HTML5
- CSS3 (with modern features like Grid, Flexbox, and animations)
- Vanilla JavaScript (ES6+)

## Usage

1. Open `index.html` in your web browser to view the website
2. Navigate through the sections using the menu or smooth scrolling
3. Click the WhatsApp icon to start a conversation (currently uses a placeholder number)
4. Use the contact information to call or email for inquiries

## Customization

- **WhatsApp Number**: Update the phone number in `index.html` (line with `wa.me/`)
- **Content**: Modify text and course information in `index.html`
- **Styling**: Adjust colors, fonts, and layout in `css/styles.css`
- **Functionality**: Add or modify JavaScript features in `js/script.js`

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge

## Deployment

This is a static website that can be deployed to any web server or hosting service like:
- GitHub Pages
- Netlify
- Vercel
- Traditional web hosting

## Contact

For inquiries about tuition services, please use the contact form on the website or connect via WhatsApp.
